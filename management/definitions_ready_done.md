# Definition of Ready / Definition of Done (baseline)

## Definition of Ready (DoR)
Работа может быть взята в исполнение, если:
1) есть ссылка на источник требований (`RS-*`) и/или решения (`D-*`);
2) зафиксированы входные данные/ограничения (например, контур калькулятора v1);
3) определены выходные артефакты (код/документ/endpoint/таблица);
4) указаны UAT сценарии, которые подтверждают результат;
5) определен владелец приемки (роль/ФИО/сторона).

## Definition of Done (DoD)
Работа считается завершенной, если:
1) изменения внесены и трассируются к `WI-*`;
2) выполнены проверки (тесты/команды) и зафиксировано доказательство;
3) обновлены сопутствующие документы (если изменились требования/контракты/приемка);
4) отсутствуют дефекты Severity 1/2 по затронутым сценариям;
5) подготовлено описание изменения для релиз-нот.
