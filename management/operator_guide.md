# Инструкция оператора и менеджера сервиса

Документ описывает ежедневную работу с лидами, эскалациями и Telegram-консультантом.
Актуальная версия приложения FastAPI - `0.1.0`, версия расчетной формулы - `v1.0`.

## Рабочие разделы

- `/admin` - сводка по лидам, диалогам, эскалациям, базе знаний и курсам валют.
- `/admin/leads` - список лидов, фильтры и CSV-экспорт.
- `/admin/calculator` - служебный расчет стоимости по формуле `v1.0`.
- `/admin/kb` - просмотр источников и добавление материалов базы знаний.
- `/docs` - Swagger UI для проверки API при необходимости.

## Вход в админку

1. Откройте `/admin`.
2. Если включен `ADMIN_API_TOKEN`, система перенаправит на `/admin/login`.
3. Введите токен администратора или оператора, выданный администратором сервиса.
4. После работы используйте `Выйти` в боковом меню.

В локальном dev-режиме пустой `ADMIN_API_TOKEN` открывает админку без входа. На shared,
stage и production окружениях такой режим использовать нельзя.

## Обработка лидов

1. Откройте `/admin/leads`.
2. Используйте фильтр по статусу и поиск по параметрам заявки.
3. Проверьте сводку параметров: модель, бюджет, страна доставки, контакт и дополнительные
   требования.
4. Для передачи данных во внешнюю CRM или таблицу нажмите `Скачать CSV`.
5. При необходимости рассчитайте ориентир стоимости в `/admin/calculator`.

Основные статусы:

- `qualified` - заявка собрана и готова к обработке;
- `budget_low` - бюджет ниже порога и требуется ручное решение;
- `new` или другие статусы - заявка требует проверки по фактическому содержимому.

## Эскалации Telegram-консультанта

Менеджер должен подключаться вручную в следующих случаях:

- клиент прямо просит менеджера, оператора, человека или специалиста;
- бюджет ниже `BUDGET_THRESHOLD_RUB`;
- запрос выходит за стандартный контур: электромобиль, мощность выше 160 л.с., страна вне
  РФ/РБ/КЗ или нестандартная схема;
- клиент отправляет фото, файл, голосовое или другое нетекстовое сообщение;
- клиент пишет после завершения квалификации;
- консультант не уверен в ответе или внешний сервис ответа недоступен.

Причина передачи фиксируется в `Escalation.reason_code`. Подробные операционные правила и
разрешенные формулировки описаны в `management/telegram_ai_consultant_skill.md`.

## Takeover менеджером в Telegram

Если Telegram-пользователь сопоставлен с ролью `manager`, его сообщение в клиентский чат
ставит бота на паузу для этого диалога. Это позволяет менеджеру вести разговор без
автоответов консультанта.

Чтобы вернуть автоответы бота, отправьте в этот чат команду:

```text
/resume_bot
```

После команды бот отвечает, что снова активен, и продолжает обработку новых сообщений.

## Расчет для клиента

1. Откройте `/admin/calculator`.
2. Используйте только поддерживаемый контур v1: РФ, возраст 3-5 лет, мощность до 160 л.с.
3. Введите суммы расходов и нажмите `Рассчитать`.
4. Перед отправкой клиенту проверьте итог и строки расчета.

Если параметры не входят в контур `v1.0`, не обещайте автоматический расчет. Передайте
кейс профильному менеджеру для ручной оценки.
