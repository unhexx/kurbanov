# Предварительный опросник для подготовки Коммерческого предложения (КП)

**Проект:** Telegram-бот для квалификации клиентов по покупке автомобилей + агрегатор логистики  
**Версия:** 2.0  
**Дата актуализации:** 13.05.2026

## 1. Фиксация входных данных (подтверждено заказчиком)

### Основной проект — Telegram-бот
- Формат взаимодействия: conversational (естественный диалог).
- Квалификация клиента: марка, год/возраст, цвет, кузов, привод, объём двигателя, топливо, пробег, КПП, ремонтные элементы, бюджет, опции, альтернативные авто, страна доставки, доп.пожелания.
- Бюджетная валидация: ниже 1 500 000 ₽ — отказ в целевом подборе + предложение альтернатив.
- База знаний: таблицы + текстовый контент (TXT/FAQ), правило «без галлюцинаций».
- Handover менеджеру: обязателен в нестандартных и ограничительных сценариях.
- Калькулятор: только РФ, авто 3–5 лет, до 160 л.с.
- Масштаб первой версии: до 10 одновременных диалогов.

### Второй проект — логистический агрегатор
- Сбор объявлений с 15 платформ.
- Цель: единый интерфейс с мощными фильтрами и экономией времени логиста.
- Стратегия интеграции: приоритет получения API, при отсутствии — парсинг.

## 2. Подтверждённые ответы на ключевые вопросы

### Общие
1. Подтверждение базовых требований: **да**.
2. Приоритет: **Telegram-бот (MVP) в первую очередь**.
3. Срок: ориентир **1 месяц**, бюджет требует оценки.
4. Приёмка: заказчик + коллеги (включая Тамару как главного логиста).

### По Telegram-боту
5. Формат базы знаний: **TXT + таблицы + docx на старте**.
6. Обновление базы: часто на старте, далее ~раз в 3–4 месяца.
7. Реальные примеры диалогов: пока не предоставлены.
8. Handover: при участии менеджера бот должен замолкать и возвращаться только по явной команде.
9. Источники авто:
   - Япония: https://aleado.com/
   - Китай: https://www.che168.com/, https://www.dongchedi.com/, https://www.yoojia.com/
   - Корея: http://encar.com/, https://www.kbchachacha.com/public/main.kbc
   - Характеристики: https://drom.ru/

### По калькулятору
10. Параметры и логика:
- Только РФ, только 3–5 лет, до 160 л.с.; иначе — менеджер.
- Вход: цена, мощность, объём двигателя, регион.
- Константы (актуализация заказчика): экспортные 15 000 ¥, комиссия 50 000 ₽, брокер 75–85 тыс ₽, доставка ~190–200 тыс ₽.
- Курсы: из админки (по данным DZENPAY).

11. Готовые формулы/таблицы:
- Есть референс XLS для 3–5 лет.
- Для других кейсов (в т.ч. 1–3 года) требуется отдельная проработка.

### По агрегатору логистики
12. Платформы: подтвержден список из 15 бирж.
13. API/парсинг: API-данные запрашиваются у площадок, проектно учитывать гибридный контур.
14. Критичные фильтры:
- дата погрузки,
- тип кузова,
- тип загрузки,
- тип оплаты,
- география (откуда/куда + радиусы),
- вес/объём (диапазоны).

### Технические и организационные
15. Хостинг: предпочтительно собственный/контролируемый контур.
16. Интеграции с CRM: желательно, окончательно не зафиксировано.
17. Оплата: обсуждаются модели 50/50 или 70/30.

5. Формальную модель ролей и доступов.
6. Согласованный SLA по обновлению данных и уведомлениям.
